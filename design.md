You are a senior UI/UX designer and frontend developer.

Design a modern, clean, and professional personal tech blog interface using Next.js and Tailwind CSS.

Design Goals:

* Minimal, distraction-free reading experience
* Professional developer portfolio aesthetic
* Fast, responsive, and accessible UI
* Focus on typography and readability

Pages to Design:

1. Homepage (blog listing)
2. Blog post page
3. Navbar + Footer
4. Optional: About page

Homepage Requirements:

* Hero section with name, tagline, and short intro
* List of blog posts in clean card or list layout
* Each post should show title, date, tags, and short description
* Subtle hover effects
* Good spacing and visual hierarchy

Blog Post Page:

* Large readable title
* Author/date info
* Clean typography (good line height, margins)
* Styled headings, lists, blockquotes
* Syntax highlighted code blocks
* Reading progress bar (optional)
* Share buttons (optional)

Navbar:

* Simple and sticky
* Logo/name on left
* Links (Home, About, GitHub, LinkedIn)
* Dark mode toggle

Footer:

* Minimal with social links and copyright

Design System:

* Use Tailwind CSS
* Font pairing: modern sans-serif (e.g., Inter)
* Consistent spacing scale
* Soft shadows, rounded corners (not excessive)
* Max width for content (avoid full-width text)

Color Theme:

* Light mode + Dark mode support
* Neutral base (white/gray/black)
* One primary accent color (blue or purple)
* Ensure good contrast for accessibility

Components to Build:

* PostCard
* TagBadge
* Navbar
* Footer
* CodeBlock (styled)
* Container layout

Extras:

* Smooth transitions and hover effects
* Skeleton loading states (optional)
* Mobile-first responsive design

Output:

* Tailwind-based component code
* Page layouts (Next.js App Router)
* Reusable components
* Clean folder structure
* No unnecessary complexity

Avoid:

* Overdesign
* Too many colors
* Cluttered UI
* Heavy animations

Focus on a clean, developer-centric aesthetic like modern SaaS or portfolio sites.
