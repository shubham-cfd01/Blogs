Welcome. This is a minimal Next.js + Markdown blog.

## Why minimal?

Less code, fewer bugs, easier to maintain. Every file in this repo is small and does one thing.

## A code sample

```ts
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```

That's it. Add more `.md` files under `/posts` and they show up automatically.
